= 註冊 NetApp Data Migrator 支援
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
註冊您的 NetApp Data Migrator 產品即可存取 NetApp 支援網站。

.步驟
. 在 NetApp Data Migrator 主頁上，選取 *說明（？）*> *關於* > *序號*
. 在「關於 NDM」對話方塊中，記錄 NetApp Data Migrator 執行個體 ID。
+
這是一個 20 位數的號碼，以 975 開頭。

. 前往 link:https://register.netapp.com["NetApp 支援網站註冊頁面"^]。
. 選取 *I am not a registered NetApp Customer* 。
. 請填寫所需的產品註冊詳細資料：
+
.. 請將 *NetApp Reference SN* 欄位留空。
.. 從產品線下拉式選單中、選取 *NDM*。
.. 從「計費提供者」下拉式選單中，選擇 *NetApp*。
+

NOTE: 本產品不收取費用。

.. 在 *NDM 序號* 欄位中輸入 20 位數的 NetApp Data Migrator 執行個體 ID。
.. 完成其餘必填欄位，然後選取 *Submit* 。


. 提交後，您將收到一封註冊確認電子郵件。請依照電子郵件中的說明確認註冊。
